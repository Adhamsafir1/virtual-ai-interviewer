Senior AWS Devops Architect with Landing Zone

Fort Worth, TX- Onsite

12 Months Contract

Job Description

The Role

We are looking for someone who can work directly with enterprise customers to design, build, and deliver cloud foundations that support large-scale platforms and migration programs. This is a consulting role with real hands-on depth. You will not just advise — you to build alongside delivery teams and own the technical outcomes.

Key Responsibilities

Design and implement AWS Landing Zones (Control Tower, AFT, multi-account strategy) with governance, networking, identity, and security baselines
Define cloud platform strategies for application modernization, database migrations, container orchestration (EKS), and hybrid architecture
Build and optimize Infrastructure CI/CD pipelines using Azure DevOps or similar (YAML pipelines, service connections, environments, approvals) with Terraform/Terragrunt as the primary IaC tool
Design Terraform module architectures including reusable modules, remote state management, workspace strategies, and drift detection for enterprise-scale AWS environments
Establish governance frameworks using AWS Service Control Policies, IAM, Cost Explorer, tagging strategies, and compliance guardrails — all deployed and enforced through CI/CD
Lead platform engineering efforts including EKS cluster design, networking (VPC, Transit Gateway, PrivateLink), and security architecture
Implement observability and monitoring strategies using CloudWatch, Datadog, or similar tooling
Guide and mentor delivery teams through complex technical decisions

Required Qualifications

8+ years in cloud infrastructure with deep hands-on experience in AWS
Expert-level Terraform skills including modules, providers, state management, workspaces, CI/CD integration, and managing large multi-environment deployments
Strong CI/CD pipeline experience (Azure DevOps YAML pipelines, GitHub Actions, or similar) for infrastructure delivery including multi-stage deployments, variable groups, release gates, and artifact management
Proven track record delivering Landing Zones at enterprise scale (multi-account strategy, hub-spoke networking, SCPs, policy-as-code via Terraform)
Strong experience with AWS core services: EKS, RDS, IAM, VPC, S3, Secrets Manager, KMS, CloudFormation/CDK
Solid understanding of Infrastructure CI/CD patterns like plan/apply workflows, automated testing, drift detection, and governed promotion across environments
Experience with containerization and Kubernetes (EKS, Helm, ArgoCD/GitOps patterns)
Ability to consult at the executive level and translate technical complexity into business decisions
Experience leading or guiding delivery teams through platform builds and migrations without being a people manager
Comfortable working in ambiguous, fast-moving enterprise environments

Good to Have

AWS certifications (Solutions Architect Professional, DevOps Engineer Professional)
Azure knowledge (Landing Zones, Azure DevOps) as a secondary cloud
Experience with EKS, service mesh (Istio), and GitOps (ArgoCD)
Database migration experience (RDS, DMS, PostgreSQL)
Background in regulated industries (pharma, financial services, manufacturing)
Experience with security scanning integration (SAST, DAST, SCA) in CI/CD pipelines
SAP on AWS migration experience

A Few Things to Note

This is not a pure advisory or slide-deck role. You will be hands-on.
This is not an operations or support role. You build foundations and move on.
This is not a single-project role. You will work across multiple enterprise customers.
